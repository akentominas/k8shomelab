# Bootstrap Home

This directory contains manual bootstrapping guides for each server.

Each per-server file includes:
- Ubuntu installation
- SSH setup
- Static IP networking configuration
- Configure sudoless

Use this README as a homepage to open the correct server preparatio guide.

## Server guides

- [00-install-ubuntu.md](00-install-ubuntu.md)
- [01-networking.md](01-networking.md)
- [02-ssh-setup.md](02-ssh-setup.md)
- [03-no-sudo.md](03-no-sudo.md)


## Usage

1. Choose the target server file from the list above.
2. Follow the step-by-step instructions:
    - Install Ubuntu
    - Enable and configure SSH
    - Configure static networking and IPs
3. Repeat for each server in your homelab.
